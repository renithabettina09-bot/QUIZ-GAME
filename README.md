# Quiz Game Application in C

## Description
This is a console-based quiz game developed using the C programming language.
The game asks multiple-choice questions, checks user answers, and calculates
the final score.

## Features
- Multiple-choice questions
- Score calculation
- Instant feedback
- Simple console interface

## Concepts Used
- Structures
- Arrays
- Functions
- Loops
- Conditional statements

## How to Run
1. Compile the program using a C compiler (gcc quiz.c)
2. Run the executable file
3. Answer the quiz questions
